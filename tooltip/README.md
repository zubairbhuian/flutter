# RichText

**If widget has tooltip property**
```dart

    IconButton(
    onPressed: () {},
    icon: const Icon(Icons.access_alarm),
    tooltip: "Alarm",
    ),
 
 ```


 **If widget has not tooltip property**
 ```dart

    Tooltip(
        message: "This is Tooltip",
        child: Container(
        height: 300,
        color: Colors.red,
        ),
    )
 
```
